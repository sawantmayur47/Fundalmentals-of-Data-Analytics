# Fundalmentals-of-Data-Analytics
In order to learn any new skill or technology, it is important to lay a strong foundations of its basics. Similarly, to get oneself acquainted with data analytics and before working on hands on data analytics projects, it is essential to learn the basics. 
One of the feature which makes Python best-suited for data analytics is the presence of its flexible and extensive open-source libraries.
This section aims at having a quick overview of the most common data analytics libraries in Python.
1. NumPy
2. Pandas
3. Matplotlib
4. Seaborn
